# 🎓 Final Year Project Management System (FYPMS)

This project is a web application developed to manage final year student projects in a college.

## Features

* Students can register, login and manage their projects
* Students can submit project proposals and upload files
* Teachers can review student projects and give feedback
* Admin can manage users and assign supervisors

## Tech Stack

**Backend:**
Node.js, Express.js, MongoDB

**Frontend:**
React.js, Redux, Tailwind CSS

## Description

This system helps in organizing and managing final year projects efficiently. It reduces manual work and makes communication easier between students, teachers, and admin.
